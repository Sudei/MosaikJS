Usage

how to include MosaikJS into your projects

1. Add libraries and dependencies


<!-- jQuery 1.11.0 -->
<script src="//code.jquery.com/jquery-2.2.3.min.js"></script>
<!-- MosaikJS libraries -->
<script src="lib/mosaikjs.js"></script>
<!-- MosaikJS stylesheet -->
<link rel="stylesheet" href="lib/mosaikjs.css">

2. Generate Markup


<div id="mosaic">
    <div class="mosaikjs-bg-project"></div>
    <div class="mosaikjs-wrapper">
        <div class="mosaikjs-list-boxes">

            ...

            <div class="case">
                <div class="case-img-container" data-cover="img/covers/sea.jpg" data-img="img/covers/sea-small.jpg"></div>
                <div class="case-infos">
                    <p class="info-title">Beyond the oceans</p>
                    <p class="info-bs">John williams' album</p>
                </div>
            </div>

            ...

        </div>
    </div>
</div>

3. Initilize your mosaic and choose your options


var mosaic = new MosaikJS({
    elementWidth: '33.3%',
    elementHeight: "210px",
    breakpoints:[
        {size: '768', elementWidth: '50%', elementHeight: "180px" },
    ],
    features: {
        'responsive': true,
    }
});

(bonus). Add Event Listener


var container = document.querySelector('#mosaic');
container.addEventListener('enter', function (e) {
    // do stuff ...
}, false);

Options

Object options, callback and events

options

elementWidth: '50%'
pixel/percent value (with suffix 'px' or '%'). This is the default, or mobile on responsive, width for each element/case.
elementHeight: '100px'
pixel/percent value (with suffix 'px' or '%'). This is the default, or mobile on responsive, height for each element/case.
breakpoints: [ {size: '768', elementWidth: '50%', elementHeight: "180px" }, ... ]
array of breakpoint. Describe elements width/height by given breakpoints.
features: { 'responsive': false }
true, false. Allow the mosaic to be responsive. A responsive mosaic use the breakpoints to rearrange element width/height. The given width should be in percent to work properly with responsive.

events

enter( element )
this event is invoked when mouse enter in a element/case.
leave( element )
this event is invoked when mouse enter in a element/case.
